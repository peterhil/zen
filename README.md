## Installation

First, clone the repository:

`git clone --recursive https://github.com/peterhil/zen.git "${ZDOTDIR:-$HOME}/.zen"`

Symlink the files to your user home directory (or `$ZDOTDIR`). This can be easily done by running:

`$HOME/.zen/install.sh`
